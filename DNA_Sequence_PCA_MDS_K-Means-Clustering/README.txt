1. Please install the packages explicilty using pip3 command because the packages aren't getting installed properly when installed from the IDE.

2. numericalConversion.py contains the python implementation of conversion of sequence into number format

4. KMeans.py performs the K-Means Clustering from scratch. The parameters are to be changed accordingly (Like the input data, centroid selection etc.,).

5. HW4.fas is the given fasta file containing the DNA sequences.
